# Contributing to Fresh People

Thank you for your interest in contributing!

## Development Setup

1. Fork the repo
2. Clone: `git clone https://github.com/YOUR_USERNAME/fresh-people-co-za.git`
3. Install: `npm install`
4. Run: `npm run dev`

## Making Changes

1. Create branch: `git checkout -b feature/your-feature`
2. Make changes
3. Run tests: `npm run test` (if available)
4. Format: `npm run format` (prettier)
5. Lint: `npm run lint` (eslint)
6. Commit: use conventional commits
7. Push & create PR

## PR Guidelines

- Fill out the PR template completely
- Link related issues
- Keep changes focused (one feature per PR)
- Update docs if needed
- All CI checks must pass

## Code Style

- Use Prettier + ESLint (auto-formatted on commit)
- Write clear, descriptive commit messages
- Comment complex logic
- Use meaningful variable names
